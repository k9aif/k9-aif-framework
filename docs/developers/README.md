
# Developers

This section is intended for developers working with the K9-AIF framework.

It will cover practical aspects of building, extending, and integrating agentic systems using K9-AIF, including:

- Setting up the development environment
- Running sample applications
- Building agents, orchestrators, and routers
- Integrating external frameworks (e.g., CrewAI)
- Working with configuration-driven flows

K9-AIF is designed as an architecture-first framework.
Developers are encouraged to review the conceptual foundation before implementation:

→ ../understanding-k9-aif/

This section will evolve as implementation patterns and examples are expanded.
