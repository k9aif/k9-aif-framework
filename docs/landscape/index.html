Dimension ⬡ K9-AIF Framework ◈ Current AI Frameworks (LangGraph / CrewAI / AutoGen etc.)
Design Philosophy Architecture-first. TOGAF-aligned. Treats multi-agent AI as an enterprise architecture discipline. Execution-first. Focus on orchestrating LLM calls and managing agent workflows. Architecture is emergent.
Orchestration Model Explicit hierarchy: Router → Orchestrator → Squads → Agents. Clear chain of command and responsibility. Varies by framework: LangGraph (state graph), CrewAI (crew/role), AutoGen (conversational), MetaGPT (SOP-guided).
Governance Built-in, cross-cutting. Governance is a first-class architectural concern embedded at every layer, not added on. Inconsistent. AutoGen & LangGraph have strong guardrails. SmolAgents has none. Most require external setup.
Building Blocks Explicit ABB/SBB separation. Architecture Building Blocks vs. Solution Building Blocks — formal structural patterns. No formal separation. Abstractions (agents, tasks, tools, memory) differ per framework and are not portable.
Model / Provider Binding Provider-independent. Unified inference and model routing layer — swap LLMs without changing business logic. Varies. Semantic Kernel & Google ADK offer abstractions. Many frameworks tightly couple to specific providers.
Configuration Configuration-driven composition. System behavior defined declaratively, enabling auditable and reproducible deployments. Code-driven. Workflows typically hardcoded (LangGraph DSL, CrewAI Python). Harder to audit without reading code.
Interoperability SOA principles. Agents exposed as services via RESTful APIs — designed for cross-system integration from the start. Siloed. Frameworks use incompatible abstractions. A CrewAI task can't directly invoke a LangGraph workflow.
Enterprise Alignment Native. Built around enterprise patterns (SOA, TOGAF, layered architecture). Designed for regulated environments. Emerging. Google ADK and Semantic Kernel have strong enterprise hooks. Most frameworks are developer/research-oriented.
Security Model Structural. Security woven into architectural layers — not bolted on after the fact. Mixed. LangGraph & AutoGen provide flow-level checks. Prompt injection, memory hijacking, supply chain risks are common concerns.
Scalability Approach Modular squads. Composable squad architecture allows horizontal scaling of specialized agent groups. Infrastructure-dependent. Relies on external deployment infra (Kubernetes, serverless) for scaling. No in-framework scaling primitives.
Maturity / Ecosystem Early-stage. Research paper published (TechRxiv). Growing but smaller community than major OSS frameworks. Mature & broad. LangChain, AutoGen, CrewAI have large communities, extensive integrations, and production deployments.
Primary Audience Enterprise architects, system integrators, regulated industries (healthcare, finance, insurance). Developers, researchers, product teams building agent prototypes through to production pipelines.
